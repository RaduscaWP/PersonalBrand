# Radu-Stefan Personal Portfolio

A modern personal developer portfolio built to present my work, skills, services, certifications, and professional identity as a Full-Stack Developer and Cybersecurity Enthusiast.

The website is designed as a serious freelance-ready portfolio, focused on clarity, performance, dark visual identity, strong typography, and real public proof through live projects and certifications.

## Live Demo

[Open Website](https://personal-brand-phi-tan.vercel.app/)

## Project Purpose

This portfolio was created to establish my personal brand as a developer and to attract freelance clients worldwide.

It presents who I am, what I build, what technologies I use, what projects I have already shipped, and what services I currently offer.

The website is not just a simple personal page. It is built as a complete professional portfolio system with multiple pages, reusable components, structured data, responsive layouts, and production-ready deployment.

## Main Features

- Modern dark personal brand design
- Fully responsive layout for desktop, tablet, and mobile
- Multi-page structure using Next.js App Router

- Custom reusable components
- Project showcase section
